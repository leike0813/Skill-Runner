## ADDED Requirements

### Requirement: 表格动作按钮 MUST 避免文本换行导致轮廓折叠
系统 MUST 在表格场景下保证动作按钮文本不换行，并保持按钮轮廓稳定。

#### Scenario: 窄视口表格按钮渲染
- **WHEN** 浏览器可视宽度较小
- **AND** 表格动作列存在操作按钮
- **THEN** 按钮文本保持单行显示
- **AND** 按钮边框不出现上下折叠错位

### Requirement: 表格动作区 MUST 使用一致布局约束
系统 MUST 为表格动作区提供统一布局约束，确保多按钮场景下对齐和间距稳定。

#### Scenario: 多按钮动作列
- **WHEN** 同一行动作列包含多个按钮
- **THEN** 按钮以一致间距展示
- **AND** 不因列宽变化出现异常叠压或样式破坏
