# Skill Runner v0.4.0

发布日期：2026-03-07  
版本类型：首个 Public Release（重大更新）

## 发布摘要
`v0.4.0` 是相较 MVP 版本的一次显著跃迁。  
本次发布不仅新增了关键能力，还对运行时架构与工程治理进行了系统性重构，显著提升了可用性、可观测性与稳定性。

## 主要更新

### 1. 新增 Interactive 执行模式
- 支持会话式执行流程，运行中可等待用户输入并继续推进任务。
- Interactive 与 Auto 模式共享统一运行时语义，行为更一致、调试更可控。

### 2. 新增 OAuth 代理鉴权与会话中鉴权
- 支持前端 OAuth Proxy 鉴权链路。
- 支持任务会话内鉴权（in-conversation auth），可在运行过程中完成鉴权并恢复执行。
- 鉴权流程与会话状态、事件流、恢复机制实现了完整打通。

### 3. 管理后端 UI 显著增强
- 管理 UI 能力覆盖更完整：引擎管理、运行观测、系统控制台等核心场景可直接操作与排障。
- 观测信息更丰富，运行态诊断与调试效率明显提升。

### 4. 架构与工程质量全面升级
- 运行时核心能力按合同（Contract）驱动，关键语义实现了更严格的一致性约束。
- 补齐并强化了测试矩阵与守护机制（约束、回归、边界防护）。
- 通过架构收敛与流程治理，整体稳定性与可维护性显著提升。

## 影响说明
- 本版本包含较大规模内部重构，建议升级后按既有部署/运行流程完成一次完整回归验证。
- 对外能力重点已从“可运行”提升到“可运行 + 可观测 + 可治理”。

感谢所有早期使用者在交互模式、鉴权链路、管理 UI、稳定性治理方面提供的持续反馈。  
`v0.4.0` 标志着 Skill Runner 正式进入可公开使用与持续迭代阶段。
