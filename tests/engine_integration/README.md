# Engine Integration Tests

This directory contains engine execution integration suites that run skill cases
through the engine integration harness fixture.

- Runner: `tests/engine_integration/run_engine_integration_tests.py`
- Wrapper: `tests/engine_integration/run_engine_integration_tests.sh`
- Suites: `tests/engine_integration/suites/*.yaml`
- Fixtures: `tests/fixtures/*`
